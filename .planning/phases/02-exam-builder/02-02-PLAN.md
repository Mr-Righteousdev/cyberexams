<task type="auto">
  <name>Task 2: Question Create Livewire Component</name>
  <files>app/Livewire/Admin/QuestionCreate.php, resources/views/livewire/admin/question/create.blade.php</files>
  <action>
Create QuestionCreate Livewire component:
- Accept exam_id from route parameter
- Properties:
  - type: 'mcq' | 'true_false' | 'short_answer' | 'code_snippet' (default mcq)
  - question_text (string)
  - marks (integer, default 1)
  - code_language (string, nullable -- for code_snippet only)
  - For MCQ: options array [{text, isCorrect}, ...]
  - For True/False: correct_answer (boolean)
  - For Short Answer: expected_keywords (string, comma-separated)
  - For Code Snippet: expected_answer (text)
- Validation rules:
  - type: required, in:mcq,true_false,short_answer,code_snippet
  - question_text: required, string
  - marks: required, integer, min:1, max:100
  - code_language: required_if:type,code_snippet
  - options (for MCQ): required_if:type,mcq, array, min:2, max:6
  - options.*.text: required
  - options.*.isCorrect: required (at least one true for MCQ)
- save() creates question, options, redirects to index

Type-specific behavior:
- MCQ: Show dynamic option rows (Add/Remove option buttons)
- True/False: Auto-create 2 options ("True", "False"), admin selects correct
- Short Answer: Just question text (no auto-grading)
- Code Snippet: Question text + language selector + expected answer

Rules:
- On save, calculate and set order = max(order) + 1 for new questions
- For True/False, auto-create options[0]=True, options[1]=False
  </action>
  <verify>
<automated>php artisan route:list --name=admin.questions.create</automated>
  </verify>
  <done>Admin can create any of the 4 question types</done>
</task>
