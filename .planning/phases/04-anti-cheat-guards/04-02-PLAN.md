**Add method to handle tab switch:**
```php
public function handleTabSwitch(): void
{
    if ($this->session->is_submitted) {
        return;
    }
    
    $this->tabSwitchCount = ($this->tabSwitchCount ?? 0) + 1;
    
    // Log to activity_logs
    ActivityLog::create([
        'session_id' => $this->session->id,
        'event_type' => 'tab_switch',
        'metadata' => [
            'count' => $this->tabSwitchCount,
            'timestamp' => now()->toIso8601String(),
        ],
        'occurred_at' => now(),
    ]);
    
    // Update session tab_switch_count
    $this->session->update(['tab_switch_count' => $this->tabSwitchCount]);
    
    // Show warning after 1st switch
    if ($this->tabSwitchCount === 1) {
        $this->showTabWarning = true;
        $this->tabSwitchWarning = 'Warning: Do not leave the exam page. Tab switches are being tracked.';
    }
    
    // Auto-submit after 3 switches (GUARD-07)
    if ($this->tabSwitchCount >= 3) {
        $this->session->update([
            'is_flagged' => true,
            'flag_reason' => 'Auto-submitted: 3 tab switches detected',
        ]);
        $this->submit();
    }
}
```

**Add method to dismiss warning:**

**Add to root div's x-data:**
Update the Alpine data to include visibility handling:
```html
<div class="mx-auto max-w-4xl select-none"
     x-data="{ 
         tabCount: {{ $tabSwitchCount }},
         showWarning: {{ $showTabWarning ? 'true' : 'false' }},
         init() {
             document.addEventListener('visibilitychange', () => {
                 if (document.hidden) {
                     // Page became hidden - student left
                     @this.handleTabSwitch();
                 } else {
                     // Page became visible again
                 }
             });
         }
     }"
     x-init="init()">
```

Actually, wire it through Livewire properly:
```html
<div class="mx-auto max-w-4xl select-none"
     x-data
     x-on:visibilitychange.window="
         if (document.hidden) {
             $dispatch('tab-switch');
         }
     "
     x-on:tab-switch.window="
         $wire.handleTabSwitch();
     "
>
```

Then in the Blade, listen for the dispatched event and call the method. Since visibilitychange doesn't give Livewire time to debounce properly, use a simple approach:
